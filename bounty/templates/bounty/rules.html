The Nitty Gritty

  • For a submission to be accepted for the bounty, you must include the phrase Darn Tootin in your screenshot with the current date & time. The following methods are accepted:
    • Using the "Clan Event" plugin on Runelite, and entering the phrase as the event password (preferred method). Be sure the date & time setting is on!
    • Typing the phrase, date,and time into chat.
  • Your screenshot must show the time achieved in the chat box right after killing the boss. The following methods will not be accepted:
    • Using the !pb command from Runelite.
    • Using the adventure log in any POH.
    • Old screenshots of personal bests.
    • Any other method of displaying times which is not listed here, but does not adhere to the above rules on submitting. Discretion will be used when verifying submissions.
  • Additional rules for the permanent leaderboard reward:
    • The permanent leaderboard reward is an additional reward, and will be added on top of whatever other reward the participant has won, if any.
    • There is no limit on how many participants can win the permanent leaderboard reward.
    • The permanent leaderboard reward will be payed out of pocket, and not subtracted from the prize pool.
  • Each participant can only win one reward, exception being the permanent leaderboard reward.
  • Submissions made for the bounty will not appear on the permanent leaderboard until the event is over.
  • The "Best Outfit" award will determined by a team of secret judges. Creativity with respect to the "Bounty" theme will be rewarded!

If you want to donate to the prize pool, please contact an admin in-game or through discord.
